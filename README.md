# Smart Temperature Monitoring System
This project monitors temperature using Arduino Uno and displays output on LCD.

## Components Used
- Arduino Uno
- Temperature Sensor 
- Resistor, Wires
- LCD Display

## Working
Sensor reads temperature → Arduino processes → LCD displays values.

## How to Run?
1. Connect components as per circuit diagram.
2. Upload Arduino code to Uno.
3. Power ON the board.
